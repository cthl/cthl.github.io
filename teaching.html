Teaching (CAAM 520 - Computational Science II)

General

Lectures are Mondays, Wednesdays, and Fridays at 1pm (location TBA). Office hours are by appointment.

Students with disabilities are encouraged to contact the Rice Disability Resource Center and to meet with me to discuss accommodations.

Prerequisites

Familiarity with C or C++, basic knowledge of Unix/Linux command line tools, Git, and GDB. Key features of C and GDB will be revisited in class.

All code and examples will be in C99, but students may use C++ in their submissions.

Students will required to create a private repository on GitHub for homework submissions (free for students, see here).

I highly recommend using a virtual machine (VM) for the homework, as the course requires software that students may find difficult to install, especially on MacOS and Windows (GCC, GDB, Valgrind, MPI, PETSc, HPCToolkit). A Linux-based VM with all required software will be provided. It can be used with VirtualBox on all major operating systems and requires about 10 GB of free disk space.

Recommended reading and reference manuals

  • The C Programming Language (2nd ed.) by Kernighan and Ritchie (1988)
  • Introduction to High Performance Computing for Scientists and Engineers by Hager and Wellein (2011)
  • MPI: A Message-Passing Interface Standard - Version 3.1
  • OpenMP Application Programming Interface
  • Debugging with GDB
